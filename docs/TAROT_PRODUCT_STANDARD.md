# FatePT 타로 상품 기준 v1 (확정)

> **이 문서의 기준은 사장님 명시 승인 시에만 변경할 수 있다.**
> 코드·DB·화면의 타로 상품 구조는 반드시 이 문서와 일치해야 한다.
> 단일 소스: `src/lib/readings/services/tarot/config.ts` — 상품명·가격·정상가·카드 수·포지션·발행 정책을 페이지마다 하드코딩하지 않는다.

## 확정 상품 기준

### 1. 원 카드 타로
- slug: `tarot-daily`
- 카드 수: **1장**
- 판매가: **1,000원**
- 발행: **자동 발행**
- 구성: 핵심 메시지와 오늘의 조언

### 2. 3 카드 타로
- slug: `tarot-inner-mind`
- 카드 수: **3장**
- 판매가: **3,800원**
- 발행: **자동 발행**
- 사용자가 다음 구성 중 선택 가능:
  - a. 과거 / 현재 / 미래
  - b. 나 / 상대방 / 관계
  - c. 원인 / 과정 / 결과

### 3. 켈틱 크로스 타로
- slug: `tarot-celtic-cross` (구 `tarot-relationship` 대체 — 5장 관계 스프레드 폐기)
- 카드 수: **10장**
- 정상가 표시: **8,700원** (프런트 취소선 표기 전용 — DB에는 저장하지 않음)
- 실제 결제 가격: **5,800원** (DB `products.price` = 5800, 주문·결제는 DB 가격만 신뢰)
- 발행: **자동 발행**
- 포지션 (순서 고정):
  1. 현재 상황
  2. 도전 또는 장애물
  3. 의식적 목표
  4. 무의식 또는 기반
  5. 과거의 영향
  6. 가까운 미래
  7. 나의 접근 방식
  8. 외부 영향
  9. 희망과 두려움
  10. 최종 결과

## 원칙

- 구 임시 가격(990 / 2,970 / 4,950)과 5장 스프레드는 폐기됐다. 저장소에 남기지 않는다.
- 기존 사주 상품·주문 데이터는 이 기준의 영향을 받지 않는다.
- 경쟁 사이트의 문장·디자인을 복사하지 않는다. 가격·구성 수치만 기준으로 삼는다.
- DB 반영은 `scripts/sql/tarot_standard_v1.sql` (타로 3행만, 주문 0건 가드 포함)로만 한다. 전체 seed 실행 금지.
