# GPT 검증 요청 패키지 — 운명PT 사주 해석/지표

> 이 파일 전체를 ChatGPT(또는 Claude)에 붙여넣고 아래 "요청"대로 검증받으세요.
> 목적: 명리학적 타당성 + 고객 언어 톤 점검. 코드 반영은 Claude Code가 합니다.

## 0. 서비스 포지션 (톤 기준)
- 브랜드: **운명PT** — "운명도 단련된다". 점술 예언이 아니라 **자기이해 트레이닝**.
- 핵심 관점: **"당신이 부족해서가 아니라, 오래 버티며 생긴 패턴이다."**
- 타깃: 주로 2030 여성. 톤: 평가 X, 해석 O. 불안 자극 → 포근한 해석 → 구체적 행동.
- 금지: 단정적 예언("~된다"), 확률 수치 단정("재회 80%"), 점술/주술 단정, 과장.

## 1. 8대 지표 점수 공식 (0~100, clamp 2~98)
아래는 실제 코드 공식입니다. `bigyeop/siksang/jae/gwan/inseong`=십성 5분류 개수,
`level`=신강 1~7(중화 4), `isStrong`=신강 여부, `ohaengCount.X`=오행 글자수,
`chungCount`=충/형/파/해/원진 수, `hapCount`=합 수, `geobjaeCount`=겁재 수,
`guiinCount`=귀인 수, `ohaengMissing`=0개 오행 목록, `hasDohwa/Hwagae/Hongyeom`=신살 유무.

```
버팀력      = 30 + (비겁+인성)*6 + (level-4)*6 + 토*4 + (신강?8:0)
자기보호력   = 32 + 비겁*7 + 충*5 + (level-4)*5 + 겁재*4
감정감지력   = 34 + (인성+식상)*6 + 수*5 + (도화?6:0) + (홍염?5:0)
관계피로도⚠ = 28 + 비겁*5 + 겁재*7 + 충*6 + (수결핍?8:0)
반복위험도⚠ = 30 + 합*6 + 결핍오행수*7 + (식상0이면 12) + (화개?5:0)
전환준비도   = 30 + (식상+재성)*6 + 귀인*3 + (대운십성있으면 6) + 화*3
돈누수감도⚠ = 26 + 비겁*6 + 겁재*7 + (재성<=1이면 10) + (비겁우세면 8)
회복탄력성   = 34 + 인성*7 + 귀인*4 + (5-결핍오행수)*4 + (신강?6:0)
```

## 2. 핵심 매핑 가정 (검증 대상)
- 비겁(비견/겁재) 과다 → 돈 누수 ↑, 관계 피로 ↑, 자기보호 ↑
- 인성 → 회복력/버팀력 ↑
- 식상 → 표현/전환/감정 출구
- 충 → 방어반응/불안정, 합 → 얽힘/반복
- 결핍 오행 → 반복 위험 ↑
- 신강 → 버팀/회복 ↑

## 3. 무료 리포트 23항목 (제목 — 잠금/무료)
[무료] 오래 버틴 사람의 기질 · 지금 반복되고 있는 신호 1가지
[나] 내 안의 진짜 동력 · 남에게 안 들키는 약한 지점 · 내가 나를 검열하는 방식 · 타고난 강점 근육 3가지 · 결핍 기운이 만든 습관
[관계] 관계에서 반복되는 자세 · 끌리는 사람/지치는 사람 · 갈등이 터지는 지점 · 거리 두기와 죄책감 · 곁에 두면 회복되는 인연
[돈·일] 돈이 새어나가는 지점 · 나에게 맞는 돈 버는 방식 · 일에서 지치는 진짜 이유 · 전환·이직 타이밍
[흐름] 2026년 전환 흐름 · 대운이 바뀌는 길목 · 조심해야 할 시기 · 움직이기 좋은 시기
[회복] 나만의 회복 루틴 · 번아웃 경고 신호 · 오늘부터의 작은 처방

## 4. 신살 → 자기이해 언어 변환 대상 (점술 단정 빼고)
역마살·도화살·화개살·망신살·재살·홍염살 등. 각 신살을 "성향/패턴" 언어로.

## 요청 (GPT가 답할 것)
1. **2번 매핑이 명리학적으로 타당한가?** 틀리거나 위험한 가정 교정.
2. **1번 공식의 가중치(계수)** 가 합리적인가? 더 정확한 계수 제안 (특히 비겁→돈누수, 인성→회복).
3. **신살 6종**을 운명PT 톤(자기이해·비단정)으로 한 줄씩 변환.
4. **CS 스크립트**: "재회 가능성 몇 %?" 질문에 단정 없이 답하는 3문장.
5. (선택) **8지표 네이밍**이 2030 여성에게 후킹되는지 + 대안.

답변은 "교정 전 → 교정 후 → 이유" 형식으로 주면 좋음.
