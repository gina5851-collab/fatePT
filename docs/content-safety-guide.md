# 콘텐츠 건강성 가이드 (Content Safety Guide)

> 외부 플랫폼(유튜브·인스타·틱톡·스레드·블로그)에서 계정 제한 없이
> 안전하게 노출할 수 있는 콘텐츠 기준을 정리합니다.
> 이 가이드는 PG 심사·광고 심사·플랫폼 정책 모두를 고려합니다.

---

## 1. 외부 플랫폼에서 피해야 할 표현

### ❌ 절대 금지 (계정 차단·광고 거절 위험)

| 피해야 할 표현 | 이유 |
|---|---|
| "사주 봐드립니다" | 미신·점술 분류 위험 |
| "점 봐드립니다" | 미신·점술 분류 위험 |
| "운명 확인" | 운명론적 단정 |
| "재회운 99%" | 수치 단정, 광고법 위반 가능 |
| "궁합운 대박" | 과장 표현, 광고법 위반 |
| "반드시 일어납니다" | 단정적 예언 |
| "이 날짜에 큰일 납니다" | 공포 조장 |
| "전남친 속마음 맞춰드립니다" | 제3자 정보 단정 |
| "외도/잠자리/임신/질병/죽음 단정" | 민감 정보 단정, 법적 리스크 |
| "올해 대박납니다" | 과장 성과 약속 |
| "이 사람 마음이 아직 있습니다" | 제3자 심리 단정 |

### ⚠️ 주의 (맥락에 따라 위험)

- "재회", "복합", "이별 후 연락" — 자사몰 내부에서는 OK, 외부 광고에는 주의
- "사주", "명식", "대운", "세운" — 상품 상세 페이지 안에서는 OK, 광고 제목/썸네일에는 주의
- "궁합" — 자사몰 내부 OK, 외부 노출 시 "관계 패턴"으로 교체

## 2. 외부 플랫폼에서 권장하는 표현

### ✅ 안전하고 효과적인 표현

| 권장 표현 | 적용 상황 |
|---|---|
| 반복 패턴 | 연애/돈/선택 모든 영역 |
| 관계 흐름 | "궁합" 대신 |
| 선택 기준 | "운명" 대신 |
| 마음 정리 | "재회" 주제 접근 시 |
| 돈이 새는 패턴 | 재물운 대신 |
| 이직 전 점검 | 직업운 대신 |
| 오늘의 선택 | 오늘의 운세 대신 |
| 자기이해 리포트 | "사주 풀이" 대신 |
| 관계 체크리스트 | "궁합 확인" 대신 |
| 감정 습관 | 연애 패턴 설명 시 |
| 흐름 분석 | "운세" 대신 |

## 3. 자사몰 내부에서만 사용 가능한 표현

결제 후 진입한 상품 페이지·결과지에서는 아래 표현을 사용할 수 있습니다.
단, 광고 소재·외부 SNS 게시물·검색 메타 설명에는 사용하지 않습니다.

- 사주 명식 / 4기둥 / 년주·월주·일주·시주
- 천간·지지 / 십성 / 대운·세운
- 오늘의 운세 (결과지 섹션 헤딩으로는 OK)
- 재회 흐름 (결과지 본문에서만)
- 궁합 (결과지 내부 분석에서만)

| 원본 (내부용) | 외부 노출용 |

## 7. 자사몰 내부 업셀 원칙

### ❌ 나쁜 업셀
- "지금 바로 프리미엄 사주 결제하세요"
- "재회운을 보면 모든 답을 알 수 있습니다"
- "이 상품을 사야 정확합니다"

### ✅ 좋은 업셀
- "오늘 관계 흐름에서 마음에 걸리는 사람이 떠올랐다면, 두 사람의 관계 패턴을 별도 리포트에서 더 자세히 정리해볼 수 있습니다."
- "오늘 돈의 흐름이 유독 신경 쓰였다면, 돈이 새는 패턴과 잡아야 할 기회를 재물 리포트에서 더 구체적으로 확인할 수 있습니다."
- "오늘의 흐름만으로는 부족하다면, 올해 전체 선택 흐름을 기본 리포트에서 이어서 볼 수 있습니다."

## 8. 검토 체크리스트 (콘텐츠 발행 전)

- [ ] 제목에 "사주/점/운명/운세" 없음
- [ ] 수치·확률 단정 없음 (99%, 반드시, 확실히)
- [ ] 제3자(전남친, 상대방 등)의 마음·행동 단정 없음
- [ ] 건강·임신·금전 성과 확정 표현 없음
- [ ] 공포 조장 표현 없음 (큰일납니다, 조심해야 합니다 → OK, "이 날짜에 큰일 납니다" → NG)
- [ ] 외부 노출 텍스트에서 CTA가 상담형·자기이해형 언어를 사용함
