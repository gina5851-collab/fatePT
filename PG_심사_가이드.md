# 토스페이먼츠 PG 심사 통과 가이드 (사주 카테고리)

> 사주·운세 카테고리는 PG 사들이 거절하기 쉬운 업종입니다.
> 토스페이먼츠 기준 실제 통과 사례를 바탕으로 정리했습니다.

## 1. 사업자 준비

| 항목 | 비고 |
|---|---|
| 사업자등록증 | **업태: 정보통신업 / 종목: 컴퓨터프로그래밍 서비스업** 권장 (점술·역술 X) |
| 통신판매업 신고 | 필수. 관할 구청 또는 정부24에서 온라인 신청 |
| 사업자 통장 | 사업자 명의 |
| 대표자 신분증 | |

## 2. 사이트 필수 요소

PG 심사관이 직접 사이트를 둘러봅니다. 다음이 **모두** 있어야 합니다:

- [x] 회사명 / 대표자 / 사업자등록번호 / 통신판매업 신고번호 / 주소 / 전화 / 이메일 푸터 노출
  → 이 보일러플레이트의 `/legal/business` 페이지가 이 역할
- [x] **이용약관 / 개인정보처리방침 / 환불정책** 별도 페이지
  → `/legal/terms`, `/legal/privacy`, `/legal/refund-policy`
- [x] 상품 상세에 **가격 / 제공 내용 / 제공 방식** 명시
  → `/products/[slug]` 자동
- [x] 결제 전 약관 동의 (토스 위젯의 AGREEMENT 영역이 자동 처리)
- [x] HTTPS 도메인 (Vercel 기본 제공)

## 3. 카테고리 표현 팁

심사 거절을 줄이는 **상품 설명 작성 가이드**:

| ✗ 피할 표현 | ✓ 권장 표현 |
|---|---|
| 점, 점술, 운명 풀이 | 사주 분석, 성향 분석 리포트 |
| 미래를 예언 | 시기별 흐름 참고 |
| 부적, 굿 | (취급 X) |
| 100% 적중 | 참고 정보 제공 |

상품 설명에 "본 서비스는 참고용 정보이며, 의학적·법률적·재정적 결정의 근거가 아닙니다" 와 같은 면책 문구 1줄 추가 권장. 이 보일러플레이트의 이용약관 제3조에 이미 포함되어 있습니다.

## 4. 결제 카테고리 신청

토스페이먼츠 가입 시 **상품 카테고리** 선택 화면에서:

- 1순위: **온라인 콘텐츠 / 디지털 서비스**
- 2순위: **컨설팅 / 컨텐츠 구독**
- ✗ 피해야 할 카테고리: 점술/역술/사주 (별도 카테고리로 노출 시)

## 5. 심사 거절 시 재신청

거절 사유는 보통 다음 중 하나:

| 사유 | 대응 |
|---|---|
| 사이트에 사업자 정보 없음 | `/legal/business` 채우기 |
| 환불 정책 모호 | `/legal/refund-policy` 의 환불 기준·절차 구체화 |
| 비과학적 단정 표현 | 상품 설명에서 "100% 적중" 등 단정 표현 제거 |
| 도메인이 IP 또는 비-HTTPS | 도메인 + HTTPS 적용 후 재신청 |

거절 메일에 적힌 사유를 그대로 수정하고 재신청하면 보통 1주일 이내 통과됩니다.

## 6. 라이브 키 교체

심사 통과 → 토스 콘솔에서 라이브 키 발급 → Vercel 환경변수의

- `NEXT_PUBLIC_TOSS_CLIENT_KEY`
- `TOSS_SECRET_KEY`

두 개를 라이브 키로 교체 → **Redeploy**.

> 💡 라이브 전환 직후 본인 카드로 최저가 상품 1건을 실결제 → 정상 승인 + 결과지 도달 확인 → 즉시 환불(토스 대시보드) 하는 것을 권장합니다.
